## Overview

This project demonstrates the implementation of an end-to-end SOC (Security Operations Center) lab using:

- Splunk Enterprise
- Splunk Universal Forwarder
- Linux Mint
- Ubuntu VM
- VirtualBox

The project includes:
- Real-time log forwarding
- SSH brute-force attack simulation
- Failed login detection
- Alert creation
- Troubleshooting and log analysis
- SSL-based secure log transmission exploration
