# End-to-End SOC Lab Implementation using Splunk Enterprise

End-to-End SOC Lab Implementation using Splunk Enterprise and Universal Forwarder

## Overview
This project demonstrates the implementation of a SOC lab using:
- Splunk Enterprise
- Splunk Universal Forwarder
- Linux Mint
- Ubuntu VM
- VirtualBox

## Features
- Real-time log forwarding
- SSH brute-force attack simulation
- Failed login detection
- Alert creation
- Troubleshooting
- SSL exploration

## Tools Used
- Splunk Enterprise
- Splunk Universal Forwarder
- Linux Mint
- Ubuntu
- VirtualBox
